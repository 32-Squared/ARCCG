# AcceleRacers CCG

Personal async multiplayer implementation of the Hot Wheels 
AcceleRacers Collectible Card Game (2004, Mattel).

## Play
Visit arccg.netlify.app

## How it works
- Single HTML file + game engine + card manifest
- Game state is encoded in the URL — share the link to pass turns
- No server, no login, no database

## Files
- index.html        — UI
- engine.js         — Game rules engine  
- card_manifest.json — All 246 cards
- cards/            — Card images (WebP)
- rules/           — Rulebook images (WebP)
- imgs/            — UI images (WebP)
